# offlinedoc

一个离线的文档生成应用，基于electron，vue

## Build Setup

``` bash
# 安装依赖
npm install

# 运行前端代码
npm run dev

# 打包前端代码
npm run build

# 启动应用
npm start

# 打包应用
npm run package
```
